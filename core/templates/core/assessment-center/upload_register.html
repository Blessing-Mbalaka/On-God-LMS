Expected CSV Format

The CSV file should contain the following columns with headers:

student_number,first_name,last_name,email,qualification,status
STD1001,John,Smith,john.smith@example.com,BSc Computer Science,present
STD1002,Jane,Doe,jane.doe@example.com,BSc Information Technology,absent
STD1003,Robert,Johnson,robert.j@example.com,BSc Data Science,present
Column Details:
  • student_number: Unique identifier for the student
  • first_name: Student's first name
  • last_name: Student's last name
  • email: Student's email address
  • qualification: Student's qualification/course (must match existing qualifications)
  • status: "present" or "absent" for the exam
Note: The qualification must exactly match an existing qualification in the system. View available qualifications
